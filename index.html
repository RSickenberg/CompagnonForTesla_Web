Security & Privacy

Security along with privacy is a fundamental human right and also a core value.
The core of this app is build with strong on-device keychain encryption using the Apple Secure Enclave and no servers are between your device and Tesla for the core features!
Companion for Tesla use Google Firebase and Instabug for debugging and analytics purpose but none of the collected data is linked to you or your Tesla account!

  • On-device encryption with hardware-based key manager
  • Anonymized data collection for debugging purposes
  • No servers between you and Tesla
  • Token based communications
